🧰 Radio Trace Extractor – Lokale Umwandlung großer JSON-Dateien

Dieses Tool dient zur lokalen Aufbereitung von sehr großen JSON-Traces (>1 GB), 
die in der Analyse-App nicht direkt verarbeitet werden können.

📦 Inhalt:
- extract_gui.py  → startet die grafische Oberfläche
- extract.py      → enthält die Umwandlungslogik
- start.bat       → Doppelklick genügt!

🛠️ So geht's:
1. Stelle sicher, dass Python 3.x auf dem Rechner installiert ist
   (Einmalige Installation durch die IT)

2. Entpacke diese ZIP-Datei

3. Doppelklicke auf `start.bat`

4. Wähle deine Quelldatei (`.json`) und Zielpfad

5. Die Datei wird aufbereitet und kann anschließend in die App geladen werden
